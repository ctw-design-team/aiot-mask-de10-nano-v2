# AIoT Mask DE10-Nano v2

AIoT Mask DE10-Nano Version 12 PCB design files.

This repository contains the hardware design package for the **AIoT Mask DE10-Nano v2**, including board layout data, schematic-related outputs, manufacturing files, libraries, and 3D models used for development and fabrication review.

## Project overview

The AIoT Mask DE10-Nano is a PCB shield / mask board intended for DE10-Nano based AIoT prototyping.  
This Version 12 package is organized for design review, manufacturing preparation, and hardware documentation.

## Repository structure

- `BOARD/` — PCB board design files
- `BOM/` — Bill of Materials
- `DATASHEET/` — component datasheets
- `FOOTPRINT/` — PCB footprint libraries
- `GERBER/` — manufacturing Gerber outputs
- `NETLIST/` — generated netlists
- `SCHEMATIC/` — schematic-related files
- `SYMBOL/` — schematic symbol libraries

- Current repository content is based on **Version 12**.

- Please verify the latest manufacturing outputs in the `GERBER/` folder before fabrication.
